Manage metadata in your documents fast and easy with our intuitive and powerful API — just a few lines of code. The guides below cover the common scenarios when files are stored on a local drive.

- [Get document info]({{< ref "metadata/python-net/developer-guide/basic-usage/get-document-info.md" >}}) — read format, MIME type, page count, size, and encryption state.
- [Find metadata properties]({{< ref "metadata/python-net/developer-guide/basic-usage/find-metadata-properties.md" >}}) — search properties by tag using a predicate.
- [Set metadata properties]({{< ref "metadata/python-net/developer-guide/basic-usage/set-metadata-properties.md" >}}) — add or update properties that match a predicate.
- [Remove metadata properties]({{< ref "metadata/python-net/developer-guide/basic-usage/remove-metadata-properties.md" >}}) — remove properties by tag, type, or value.
- [Clean metadata]({{< ref "metadata/python-net/developer-guide/basic-usage/clean-metadata.md" >}}) — strip every detected property with a single call.
