The advanced guides cover loading and saving options, working with metadata properties and standards, and other techniques beyond the basics.

{{< alert style="info" >}}
Runnable versions of every example live in the [GitHub examples repository](https://github.com/groupdocs-metadata/GroupDocs.Metadata-for-Python-via-.NET/).
{{< /alert >}}

- [Loading files]({{< ref "metadata/python-net/developer-guide/advanced-usage/loading-files/_index.md" >}}) — load from disk, a stream, a specific format, or a password-protected document.
- [Saving files]({{< ref "metadata/python-net/developer-guide/advanced-usage/saving-files/_index.md" >}}) — save to a path, a stream, or back to the original source.
- [Working with metadata properties]({{< ref "metadata/python-net/developer-guide/advanced-usage/working-with-metadata-properties/_index.md" >}}) — extract property values and export the metadata tree.
- [Working with metadata standards]({{< ref "metadata/python-net/developer-guide/advanced-usage/working-with-metadata-standards/_index.md" >}}) — read and edit EXIF, IPTC IIM, and XMP metadata.
- [Extracting metadata]({{< ref "metadata/python-net/developer-guide/advanced-usage/extracting-metadata.md" >}}) — search by tags and categories.
- [Adding metadata]({{< ref "metadata/python-net/developer-guide/advanced-usage/adding-metadata.md" >}}) — add known-but-missing properties.
- [Removing metadata]({{< ref "metadata/python-net/developer-guide/advanced-usage/removing-metadata.md" >}}) — remove properties by category or other criteria.
- [Getting known property descriptors]({{< ref "metadata/python-net/developer-guide/advanced-usage/getting-known-property-descriptors.md" >}}) — inspect the properties a package supports.
- [Working with interpreted values]({{< ref "metadata/python-net/developer-guide/advanced-usage/working-with-interpreted-values.md" >}}) — read human-readable values for numeric/enum properties.
