GroupDocs.Metadata provides dedicated APIs for the major image metadata standards. Each package is reached from the root package (`exif_package`, `iptc_package`, `xmp_package`) and can be read, edited, or removed.

- [Working with EXIF metadata]({{< ref "metadata/python-net/developer-guide/advanced-usage/working-with-metadata-standards/working-with-exif-metadata.md" >}}) — camera, IFD, and GPS tags.
- [Working with IPTC IIM metadata]({{< ref "metadata/python-net/developer-guide/advanced-usage/working-with-metadata-standards/working-with-iptc-iim-metadata.md" >}}) — envelope and application records.
- [Working with XMP metadata]({{< ref "metadata/python-net/developer-guide/advanced-usage/working-with-metadata-standards/working-with-xmp-metadata.md" >}}) — XMP Basic, Dublin Core, Photoshop, and custom schemes.
