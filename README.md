# GroupDocs.Metadata - Documentation Source Files

This repository contains Markdown files used to generate [GroupDocs.Metadata](https://docs.groupdocs.com/metadata/) static documentation.

## About GroupDocs.Metadata

[GroupDocs.Metadata](https://products.groupdocs.com/metadata/) is a metadata management SDK that enables you to view, add, edit, and remove metadata properties in documents, images, audio files, and other file formats.

Available for .NET, Java, Node.js, and Python.

## Repository Purpose

The files in this repository are used to:

- Build and maintain the official [documentation](https://docs.groupdocs.com/metadata/).
- Ensure documentation is always up-to-date with the latest product release.
- Provide a consistent structure for automated builds and publication.

## How to Use

This repository is not meant for direct use, as it contains source files for generating static content with Hugo.
